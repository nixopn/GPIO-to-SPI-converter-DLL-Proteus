# SPI Bridge Converter (GPIO to SPI)

Модель конвертера, преобразующего сигналы с пинов общего назначения в интерфейс SPI. Разработана для среды моделирования Proteus.

## Распиновка

| Пин | Назначение |
|-----|------------|
| **D0–D7** | MOSI. Через них передаются данные и устанавливаются настройки. MSB, то есть D0 — самый старший разряд. |
| **CLK** | Тактовый сигнал. |
| **MODE** | Установка режима. |
| **STROBE** | Импульсный сигнал передачи данных. |
| **CS** | Chip select. |
| **A0–A2** | Адрес используемого устройства. A0 — самый старший разряд. |
| **D12–D19** | MISO. D12 — самый старший разряд. |
| **INT** | Прерывание. |
| **RST** | Сброс всех настроек и данных в конвертере. |
| **VCC** | Питание. |
| **GND** | Заземление. |
| **SPI_MOSI** | Прошедший через bit-banging поток битов из D0–D7. |
| **SPI_MISO** | Поток битов от ведомого. |
| **SPI_SCLK** | Изменённый CLK. |
| **SPI_CS0–SPI_CS7** | Chip select. |

## Режимы работы (MODE / STROBE)

| MODE | STROBE | Режим |
|------|--------|-------|
| 0 | 0 | Пассивный режим |
| 0 | 1 | Передача данных конвертеру |
| 1 | 0 | Просмотр параметров |
| 1 | 1 | Установка значений параметров |

## Сигнал INT

| INT | Состояние |
|-----|-----------|
| 0 | Данных нет или они не готовы |
| 1 | Данные готовы, ведущий должен прочитать |

## Сигнал RST

| RST | Состояние |
|-----|-----------|
| 0 | Сброс |
| 1 | Стандартная работа |

## Установка значений параметров и параметры

- **D0–D3** — значение параметра
- **D4–D7** — адрес регистра параметров

## Регистры параметров

### Регистр 0000 — включение и прерывания

Значения параметров в регистре:

| Значение | Функция |
|----------|---------|
| 0001 | Включение конвертера |
| 0010 | Включение прерываний (после отправления пакета будет INT = 1) |
| 0011 | Включение конвертера в режиме прерываний |

### Регистр 0001 — деление CLK

### Регистр 0010 — четыре режима SPI

| Комбинация | Режим | CPOL | CPHA |
|------------|-------|------|------|
| 0000 | 0 | 0 | 0 |
| 0001 | 1 | 0 | 1 |
| 0010 | 2 | 1 | 0 |
| 0011 | 3 | 1 | 1 |

- **0000** — Тактовый сигнал в состоянии покоя — низкий уровень. Данные выбираются по нарастающему фронту тактового сигнала, изменение данных — по спадающему фронту.
- **0001** — Тактовый сигнал в состоянии покоя — низкий уровень. Данные выбираются по спадающему фронту, изменение данных — по нарастающему фронту.
- **0010** — Тактовый сигнал в состоянии покоя — высокий уровень. Данные выбираются по спадающему фронту, изменение данных — по нарастающему фронту.
- **0011** — Тактовый сигнал в состоянии покоя — высокий уровень. Данные выбираются по нарастающему фронту, изменение данных — по спадающему фронту.

### Регистр 0011 — размер пакета

| Значение | Размер |
|----------|--------|
| 0000 / 0001 | 1 байт |
| 0010 | 2 байта |
| 0011 | 3 байта |
